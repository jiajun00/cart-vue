# sls-vuex2-demo

> 这是用vuex2实现的一个商城购物车demo,存储数据用的本地localstorage

## 演示

### GitHub：[https://github.com/jiajun00/cart-vue/dist/](https://github.com/jiajun00/cart-vue/dist/)

## 操作步骤

``` bash
# 克隆项目
git clone https://github.com/sailengsi/sls-vuex2-demo.git

# 安装依赖
npm install

# 启动server
npm run dev

# 编译发布
npm run build
```
